# Station Challenge 2026 — QUIP Only

Everything runs inside QUIP. No website, no hosting, no external tools.

## 3 QUIP Documents to Create

1. **Station Challenge 2026 — Main Hub** (copy from `1-MAIN-HUB.md`)
   - The landing page associates see
   - Rules, scoring, links to prediction sheet

2. **Prediction Sheet** (spreadsheet — copy from `2-PREDICTION-SHEET.tsv`)
   - Associates type their predictions directly here
   - One row per person per match
   - Protected columns for admin scoring

3. **Leaderboard** (spreadsheet — copy from `3-LEADERBOARD.tsv`)
   - You update this after each match day
   - Sorted by points, split by shift
   - Shows top 5 night, 3 early, 2 late winners

## How It Works

1. Share the Main Hub link (or QR code to the QUIP link) with all associates
2. Associates open the Prediction Sheet and add their rows
3. After each match, you fill in actual results and points
4. Copy top scorers to the Leaderboard
5. At the end of the World Cup, top scorers per shift win

## Anti-Fraud

- QUIP tracks who edited what (built-in revision history)
- Each associate types their own login — QUIP logs their Amazon identity
- You can see exactly who added/edited each row and when
- Lock rows after match kickoff by changing Status to "locked"
