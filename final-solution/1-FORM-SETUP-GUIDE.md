# Microsoft Forms — Setup Guide

## Creating a Match Day Form

You'll create one form per match day (or per match week). Here's how:

### Step 1: Create the Form

1. Go to **https://forms.office.com**
2. Click **+ New Form**
3. Title: `⚽ Station Challenge — Match Day 1 (June 11)`
4. Description: `Predict the scores for today's World Cup matches. One submission per person. Predictions lock at kickoff!`

### Step 2: Add Questions

**Question 1: Amazon Login** (required)
- Type: **Text**
- Question: `Your Amazon Login`
- Subtitle: `e.g. jsmith`
- Toggle: **Required**

**Question 2: Full Name** (required)
- Type: **Text**
- Question: `Your Full Name`
- Toggle: **Required**

**Question 3: Shift** (required)
- Type: **Choice**
- Question: `Your Shift`
- Options:
  - Night Shift 🌙
  - Early Shift 🌅
  - Late Shift 🌇
- Toggle: **Required**

**Then for EACH match on that day, add a Section + 2 questions:**

**Section Header:** `🏟️ Match GS-A1: USA vs TBD-A2`
Subtitle: `MetLife Stadium, NJ — June 11, 18:00`

**Question 4: Score prediction for Team A**
- Type: **Text**
- Question: `USA — Your predicted score`
- Subtitle: `Enter a number (e.g. 2)`
- Toggle: **Required**

**Question 5: Score prediction for Team B**
- Type: **Text**
- Question: `TBD-A2 — Your predicted score`
- Subtitle: `Enter a number (e.g. 1)`
- Toggle: **Required**

**Repeat the Section + 2 questions for each match on that day.**

### Step 3: Form Settings

Click the **⚙️ Settings** (top right) and configure:

- ✅ **Record name** — captures their Microsoft identity
- ✅ **One response per person** — prevents duplicate submissions
- ✅ **Start date / End date** — set end time to match kickoff
- ✅ **Shuffle questions** — OFF
- ✅ **Show progress bar** — ON

### Step 4: Share

1. Click **Collect responses** (Send button)
2. Choose **Anyone with the link can respond**
3. Copy the link
4. Generate a QR code for the link (use any QR generator)
5. Print and post at the station

## Tips

- Create forms 1-2 days before the match day
- Close the form at kickoff time
- You can duplicate a form and just change the match details for speed
- The QR code can stay the same if you reuse the form link (just update questions)
