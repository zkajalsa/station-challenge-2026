# Complete Setup — 15 Minutes

## Step 1: Create the First Microsoft Form (5 min)

1. Go to **https://forms.office.com**
2. Click **+ New Form**
3. Title: `⚽ Station Challenge — Match Day 1 (June 11)`
4. Add these questions:

**Q1** (Text, Required): `Your Amazon Login`
**Q2** (Text, Required): `Your Full Name`
**Q3** (Choice, Required): `Your Shift` → Night Shift 🌙 / Early Shift 🌅 / Late Shift 🌇

**Add Section:** `🏟️ GS-A1: USA vs TBD-A2 — MetLife Stadium, June 11 18:00`
**Q4** (Text, Required): `USA — Your predicted score`
**Q5** (Text, Required): `TBD-A2 — Your predicted score`

**Add Section:** `🏟️ GS-A2: TBD-A3 vs TBD-A4 — SoFi Stadium, June 11 21:00`
**Q6** (Text, Required): `TBD-A3 — Your predicted score`
**Q7** (Text, Required): `TBD-A4 — Your predicted score`

5. Settings ⚙️:
   - ✅ One response per person
   - Set end date to June 11 17:30 (30 min before first kickoff)

6. Click **Collect responses** → Copy link → Generate QR code

## Step 2: Create the QUIP Leaderboard (3 min)

1. Open QUIP → **+ New** → **Document**
2. Name: `⚽ Station Challenge 2026 — Live Leaderboard`
3. Copy everything from `final-solution/3-QUIP-LEADERBOARD.md` → Paste
4. Share: **Anyone at Amazon with the link can view**

## Step 3: Print & Post (2 min)

1. Generate QR code for the Microsoft Form link
2. Print the QR code
3. Post at station entrance, break room, standup area
4. Also share the QUIP leaderboard link on Chime

## Step 4: After Each Match Day (5 min)

1. Open the Form → **Responses** → **Open in Excel**
2. For each prediction row, calculate points:
   - Exact score = **5 pts**
   - Correct winner = **2 pts**
   - Wrong = **0 pts**
3. Tally total points per person
4. Update the QUIP leaderboard with new rankings
5. Post an update on Chime: "Leaderboard updated! Check who's on top 🏆"

## Step 5: For Next Match Day

1. Duplicate the form (or create new)
2. Update match details
3. Same QR code process
4. Repeat until the Final on July 19

## That's It!

- **Microsoft Forms** = clean mobile experience for associates
- **Excel** = automatic data collection + scoring
- **QUIP** = visual leaderboard everyone can see
- **QR Code** = easy access, scan and predict

No servers. No APIs. No hosting. All Amazon-internal tools.
